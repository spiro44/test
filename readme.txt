вариант 1:
способ хранения: словарь (dict python)
строки так же словари( можно сделать объектами)
при запросе клиента сортируются ключи по выбранному атрибуту(в программе - по ключу)
по отсортированному списку ключей формируется словарь для отправки клиенту, с учетом текущей позиции/размер "окна" клиента/столбца сортировки
протокол обмена:
после установки получения от клиента текущих параметров отображения (текущая позиция/размер "окна" клиента/столбец сортировки)
запуск цикла передачи клиенту сообщений с выборкой с определенной частотой/либо по изменению состояния данных 
при получении от клиента запроса на изменение данных, выдача данных с новыми параметрами
не реализовано в скрипте:
ограничение на размер данных 100000 строк, далее скрипт упирается в изменение key словаря во время выборки
упрощенный протокол обмена client-server
